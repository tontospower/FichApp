# FichApp
Test application to learn Angular + Ionic + CSS +...?

git clone https://github.com/tontospower/FichApp.git

### Install
```bash
npm install
npm install -g @angular/cli
npm install -g @ionic/cli
```

### Run
```bash
ionic serve --external
```

### Build
```bash
ionic build --prod
```

### Deploy
```bash
ng deploy
```
